# Codex、GitHub 与 Claude Code 的区别

## GitHub 是什么

把代码存在上面的平台，类似 Google Docs + 百度网盘的合体。多人能协作编辑，改动有记录，能随时回退。

## 开源是什么

"我把菜谱公开了，你随便看、随便用、甚至可以在我的基础上改良。" 代码公开在 GitHub 上，别人不用付钱、不用打招呼就能拿去用或修改。

## Codex / GitHub Copilot 是什么

OpenAI 做的代码生成 AI，通过 GitHub Copilot（编辑器插件）提供。你写代码时它自动补全下一行或下一段，就像一个坐在你旁边的老程序员，时刻盯着你的屏幕给你递代码。按月收费（会员制），不限使用量。

## Claude Code 是什么

能接完整任务的 AI 编程助手。你用人话描述需求，它自己找文件、改代码、跑测试，搞完向你汇报。按使用量收费（像打车）。

## 三者关系

| 比喻 | 对应的东西 |
|------|-----------|
| 图书馆 | GitHub，里面放了无数公开的代码 |
| 图书馆里的书可以随便借阅 | 开源 |
| 一个读过图书馆大部分书的助手 | Codex（AI 模型） |
| 助手坐在你旁边帮你补全代码 | Copilot |
| 一个能独立接活干的包工头 | Claude Code |

## Copilot 和 Claude Code 的区别

| | Copilot | Claude Code |
|------|:--:|:--:|
| 交互方式 | 编辑器里自动弹出来，零延迟 | 聊天交互，需要停下来沟通 |
| 一次写多少 | 一两行到一小段 | 整个文件、整个功能 |
| 跨文件能力 | 只看当前文件 | 整个项目都读 |
| 改已有代码 | 不擅长 | 擅长 |
| 适用人群 | 会编程、知道自己要写什么的人 | 不一定是程序员也能用 |
| 收费模式 | 包月会员制 | 按用量计费 |
| 类比 | 电动车辅助驾驶 | 叫代驾 |

## 总结

- **写代码 → 存代码 → 跑代码 → 得到工具**，GitHub 管"存"这一步
- Copilot 适合懂编程、有想法且能自己写的人，帮你写得快
- Claude Code 适合想要直接搞定结果的人，帮你省掉整个思考过程
